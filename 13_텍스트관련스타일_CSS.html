텍스트 관련 스타일(text~)

color : 텍스트의 색상을 지정

        선택자{
            color:색상명/16진수/rgb, rgba, hsl..
        }

text-decoration : 텍스트에 줄을 긋거나 없앨때 사용

        선택자{
            text-decoration : none/underlint/overline/line-through;
        }
  • underline : 텍스트 영역 아래에 줄 긋기
  • line-through : 텍스트 영역 중간에 줄 긋기
  • overline : 택스트영역 위에 줄 긋기

text-transform : 영문 텍스트의 대소문자 변환시 사용

        선택자{
            text-transform:uppercase/lowercase/capitalize;
    uppercase : 모든 영문자를 대문자로 lowercase : 모든 영문자를 소문자로 capitalize : 영문자의 첫 글자만 대문자로

오른쪽정렬

속잎나고, 넣는 수 있는 원대하고, 못하다 쓸쓸하랴? 구하기 인간의 사는가 트고, 약동하다. 두손을 그들의 역사를 청춘의 쓸쓸하랴? 사라지지 하였으며, 뭇 이 소담스러운 공자는 인간의 있는가? 이것을 이상은 청춘에서만 미묘한 황금시대의 이상, 사막이다. 그들은 보내는 있을 않는 얼음과 같이 바이며, 운다. 뜨고, 같은 아니더면, 보라. 풍부하게 이 아니한 이것이다. 투명하되 눈이 착목한는 것이다. 못하다 위하여, 그들은 인간의 인생을 풍부하게 없으면 우는 일월과 봄바람이다. 같은 보는 구할 뭇 사막이다. 인생을 설레는 없는 시들어 그들의 것이다. 갑 가지에 뜨거운지라, 보는 우는 있으랴? 끝까지 이상, 꽃이 거친 오아이스도 같은 가장 석가는 피다. 같이 충분히 작고 귀는 품에 풍부하게 있으랴? 속에서 가치를 물방아 피다. 우리 얼마나 꾸며 발휘하기 아름답고 싶이 인간의 있으며, 말이다. 따뜻한 청춘에서만 우리의 그들은 못할 찾아다녀도, 얼마나 이상 얼마나 것이다. 이것을 너의 그들은 지혜는 것은 있는가? 안고, 열락의 지혜는 기쁘며, 밝은 내려온 것이다.

가운데정렬

그들에게 품고 있음으로써 끝까지 천자만홍이 황금시대의 소금이라 부패뿐이다. 같은 스며들어 있는 실로 사막이다. 설레는 시들어 것이 인간은 위하여서 그들은 공자는 찾아 것이다. 속잎나고, 청춘 못하다 뼈 것이다. 그들의 어디 물방아 것이다. 피어나는 불어 청춘의 황금시대를 주는 부패를 방황하였으며, 목숨이 우리 듣는다. 갑 튼튼하며, 얼마나 꽃 그들은 창공에 굳세게 그와 것이다. 넣는 청춘이 얼마나 예수는 같으며, 그들의 인간에 봄바람을 싹이 말이다. 위하여서, 뜨고, 내려온 곧 영원히 같이, 사막이다.

text-shadow : 텍스트에 그림자효과를 줄때 사용

        선택자{
            text-shadow:가로거리(x) 세로거리(y) 번짐정도 색상;
        }

        - 번짐정도와 색상은 생략 가능
        - 가로거리와 세로거리가 양수인 경우 오른쪽 아래에 그림자 생성
        - 가로거리와 세로거리가 음수인 경우 왼쪽 위에 그림자 생성
        - 번짐정도가 0인 경우 그림자 생기 진하다
        - 색상은 기본값이 글자색과 같다
